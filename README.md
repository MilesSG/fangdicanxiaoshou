# 🏢 房地产销售管理系统

## 项目简介

基于Spring Boot和MyBatis-Plus开发的房地产销售管理系统，提供房产信息展示、客户管理、销售记录等功能。

## 文档指引

- 详细使用说明文档请查看：[使用说明文档](/docs/使用说明文档.md)

## 快速开始

1. 确保MySQL服务已启动
2. 创建数据库：`fangdicanxiaoshou`
3. 配置数据库连接（默认用户名：root，密码：888999）
4. 使用Maven命令启动项目：`mvn spring-boot:run`
